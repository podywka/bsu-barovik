## Описание проекта
Разработка модуля справочной информации для бизнес-приложения с двумя взаимосвязанными справочниками.

## Справочники
1. **Справочник городов Беларуси** - содержит информацию о городах
2. **Справочник промышленных предприятий** - содержит информацию о предприятиях с привязкой к городам

## Технологии
- Язык программирования: Python
- СУБД: SQLite
- GUI: Tkinter / CustomTkinter
